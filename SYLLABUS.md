# SYLLABUS - AI Engineering Self-Degree

**Version:** 1.0 (April 27, 2026)  
**Duration:** 3–6 months (10–20 hrs/week)

---

## Phase 00: Prerequisites (1–2 weeks)
- Python mastery (OOP, APIs, async)
- Git & terminal fluency
- Basic ML concepts
- **Deliverable:** Set up repo + environment

## Phase 01: LLM Fundamentals (2–3 weeks)
- Transformers, tokenization, attention
- Key papers & from-scratch implementations
- **Key Resource:** Karpathy — LLMs from Scratch
- **Project:** Tiny GPT / character-level model

## Phase 02: Prompting, RAG & Tools (2–3 weeks)
- Advanced prompting, Chain-of-Thought
- Retrieval-Augmented Generation
- Tool calling & function calling
- **Project:** RAG chatbot over personal documents

## Phase 03: Building Agents (3–4 weeks) ← Core Focus
- ReAct, Plan-and-Execute, Reflexion
- Memory systems, multi-agent orchestration
- Evaluator-Optimizer patterns
- **Key Resource:** Anthropic "Building Effective Agents"
- **Projects:** Research agent, multi-step agent, autonomous worker

## Phase 04: Production & Systems (3–4 weeks)
- Evaluation frameworks (RAGAS, LLM-as-Judge)
- Observability, cost/latency optimization
- Deployment (FastAPI, Streamlit, LangGraph)
- **Project:** Production-grade agent with monitoring

## Phase 05: Capstone & Specialization (4+ weeks)
- Choose specialization (Multi-agent systems, Fine-tuning, AI Products, etc.)
- Large final project + public presentation
- Portfolio polish & open-source contributions

---

## Weekly Rhythm
1. Study core material
2. Complete exercises
3. Build mini-project
