1. `cd ai-engineering`
2. `python -m venv venv && source venv/bin/activate` (Windows: `venv\Scripts\activate`)
3. `pip install -r requirements.txt`
4. Read `SYLLABUS.md` and start **Phase 01**
5. Ask Grok for guidance at any time
